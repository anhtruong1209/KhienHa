# Khien Ha

Ban gui cho khach chi can zip thu muc `KhienHaFull/`.

Frontend da build san vao `KhienHaFull/public`, nen ben nhan khong can chay Next hay Node.

## Ban giao nhanh

1. Giai nen thu muc `KhienHaFull`
2. Copy `KhienHaFull/.env.quickstart.example` thanh `KhienHaFull/.env`
3. Sua nhom `DB_*` trong `KhienHaFull/.env`
4. Double click `KhienHaFull/import-db.bat`
5. Trong ServBay, tro `Website Root Directory` vao `KhienHaFull/public`

Tai khoan admin mac dinh:

```text
admin@khienha.vn / KhienHa@123456
```
